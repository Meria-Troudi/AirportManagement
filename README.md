# Airport Management System

A modular .NET 8.0 solution for managing airport operations (flights, passengers, planes, tickets) with Entity Framework Core.

## Structure
- **AM.ApplicationCore**: Domain models, interfaces, business logic
- **AM.Infrastructure**: Data access, EF Core configurations, migrations
- **AM.UI.Console**: Console interface

## Usage
1. Open `AirportManagement4ARCTIC4.sln` in Visual Studio
2. Restore NuGet packages
3. (Optional) Run `Update-Database` for migrations
4. Set `AM.UI.Console` as startup and run
